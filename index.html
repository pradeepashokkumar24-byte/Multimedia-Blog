📚 Tech Glossary

HTML
HTML (HyperText Markup Language)   is the standard markup language used to create and structure content on the web. It is the "skeleton" of every website, defining where headings, paragraphs, images, and links appear before they are styled with CSS or made interactive with JavaScript.
CSS
CSS stands for Cascading Style Sheets, a stylesheet language used to define the visual presentation of documents written in markup languages like HTML. While HTML provides the structure of a webpage, CSS controls everything from colors and fonts to layout and animations.
Java Script
JavaScript (JS) is a high-level, multi-paradigm programming language that is a core technology of the World Wide Web, alongside HTML and CSS. While historically used to make webpages interactive, it is now widely used for server-side development, mobile apps, and more.
